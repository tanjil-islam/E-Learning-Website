## ✨ Features

- **Responsive Design:** Mobile-first, fully responsive across all screen sizes
- **Smooth Animations:** Scroll animations powered by Framer Motion
- **Modern UI:** Clean, professional design with Tailwind CSS
- **Multiple Sections:** Hero, Courses, Testimonials, Benefits, FAQ, Footer
- **Performance Optimized:** Fast loading with Vite build tool
- **Interactive Elements:** Smooth scroll, hover effects, and dynamic content
- **SEO Optimized:** Semantic HTML structure for better search visibility

---

## 💻 Tech Stack

- **Frontend:** React.js, JavaScript (ES6+), HTML5
- **Styling:** Tailwind CSS
- **Animations:** Framer Motion (Motion Library)
- **Build Tool:** Vite
- **Hosting:** Vercel

## 🛠 Installation

1. Clone the repository
   ```bash
   git clone https://github.com/tanjil-islam/E-Learning-Website.git
   cd E-Learning-Website
   ```

2. Install dependencies
   ```bash
   npm install
   ```

3. Start development server
   ```bash
   npm run dev
   ```

   Open [http://localhost:5173](http://localhost:5173) in your browser.

4. Build for production
   ```bash
   npm run build
   ```
